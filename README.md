# 🛍️ Script para leitura e organização de Notas Fiscais

# Sobre o projeto

Este projeto nasceu com o intuito de automatizar uma tarefa manual onde era separado notas fiscais por número da nota e depois armazenadas em arquivo morto. 
Um tarefa que anteriormente exigia muito molha dedo e paciência, hoje as notas ficais são apenas scaneadas e ao executar o script, é feito a leitura dessas notas,
identificado o número da nota e separado em pastas conforme os três ou dois primeiros números da nota. Isso proporcionou agilidade e eficiencia no processo que 
é realizado pelo setor financeiro da empresa em que trabalho.

#  🛠️ Tecnologias Utilizadas
- Python
- pytesseract


# ▶️ Como executar o projeto

## Clonar o repositório
```bash
git clone git@github.com:DyhegoS/invoices_separation.git
```

## Entre na pasta do projeto clonado
```bash
cd invoices_separation
```

## execute o projeto

```bash
py main.py
```
